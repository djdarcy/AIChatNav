# Keyboard Shortcuts

All shortcuts use the Alt modifier key.

| Shortcut | Action |
|----------|--------|
| Alt+K / Alt+Up | Navigate to previous message |
| Alt+J / Alt+Down | Navigate to next message |
| Ctrl+Alt+K / Ctrl+Alt+Up | Jump to first message |
| Ctrl+Alt+J / Ctrl+Alt+Down | Jump to last message |
| Alt+M | Cycle mode (User -> AI -> Both) |
| Alt+H | Toggle panel visibility |
